# {{ .AppName }} - Web + Auth Starter

This project was scaffolded using the **web-auth** starter kit from GoW.

## What's Included

- Full session-based authentication (login, register, logout)
- Blade-like templating with layouts and components
- Eloquent-style ORM
- Form Request validation
- CSRF protection
- Basic user model and migrations

## Getting Started

```bash
go run main.go
# or
gow serve
```
