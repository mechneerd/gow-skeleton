# Docker Support for GoW Projects

This kit adds Docker and docker-compose support to any GoW starter.

## Files Included

- `Dockerfile`
- `docker-compose.yml`
- `.dockerignore`

## Usage

```bash
docker-compose up --build
```

## Status
**In Progress** - Basic Docker files are being added.

## Recommended Combination
Use with `full` or `web-auth` for a production-like local development environment.
