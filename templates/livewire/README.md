# Livewire-style Starter Kit

For developers who want a reactive, Livewire-like experience in pure Go with minimal JavaScript.

## What's Included (Now Real!)
- GoW Livewire fully wired (`public/js/livewire.js`)
- Reactive components (see `app/Livewire/Counter.go`)
- Beautiful modern landing page at `/` using GoBlade
- Example of wire:click and real-time updates

## Quick Start
```bash
gow make:view dashboard
# Then create components in app/Livewire/
```

## Status
**Active** — Real GoW Livewire implementation is now included.

This kit is a great starting point for dashboards and internal tools that need rich interactivity without a heavy frontend framework.
