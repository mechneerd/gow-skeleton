# {{ .AppName }} - Web Starter

Full web application with Blade-style templating.

## Getting Started

```bash
go run main.go
```

Use the view engine, create controllers, and enjoy the Laravel-like experience.

### New in this starter

- Beautiful new landing page at `/`
- GoW Livewire included for reactive components

Create views with:

```bash
gow make:view home
gow make:view components/button
```
